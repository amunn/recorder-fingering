\section*{Usage}
This package provides a single command |\fingering[<size>]{<note>}|  which will produce a fingering diagram for any note in the playable range of the recorder. The notes range from C–Eb for recorders in C (the default) or F–Ab (the |*| version of the command) for recorders in F.

Notes are noted as C-B (first octave) c-b (second octave) and c'–eb' (third octave) with an equivalent scheme starting at F for the recorders in F.

The chart below shows all of the possible fingerings.
